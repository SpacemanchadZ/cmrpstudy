# CMRPstudy

Free CMRP certification prep resource for maintenance and reliability professionals.

Live site: [cmrpstudy.com](https://cmrpstudy.com)

## What's inside

- Complete CMRP study guide covering all 5 SMRP domains and 5 Pillars of Reliability
- 6 working reliability calculators (MTBF/MTTR, OEE, Weibull, Series/Parallel, R(t), EOQ)
- 80+ term searchable glossary
- Animated CMRPstudy logo mark

## Stack

Pure HTML/CSS/JS — no build tools, no dependencies, no frameworks. One file deploys anywhere.

## Deploying

This site is deployed via GitHub Pages. Any push to `main` goes live automatically.
